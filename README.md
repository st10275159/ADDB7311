ADDB7311
Summative Project 

This repository contains a complete relational database implementation for the BikesRUS system as part of the ADDB7311 summative project.
The project demonstrates core database design and SQL programming skills using Oracle/PL-SQL.

//Technologies used 
- Oracle SQL / PL-SQL  
- Table creation with constraints  
- Referential integrity with foreign keys  
- Data insertion scripts  
- Queries involving joins, aggregations, and conditions  
- Views, cursors, procedures, functions, and triggers

//structure

- `BikesRUS3.sql` — Main SQL script including:
  - Table creation (VOLUNTEER, DONOR, BIKE, DONATION)
  - Sample data inserts
  - Select queries for reporting
  - Stored procedures and functions
  - Views
  - Business logic triggers
